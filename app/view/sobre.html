Sobre a Aplicativo para Alocação de Contratos (APLAC)

Na conjuntura atual do mercado brasileiro de energia elétrica existem dois ambientes de contratação: regulado e livre. Consumidores potencialmente livres podem escolher a procedência de energia elétrica entre os dois ambientes. A tomada de decisão de escolha entre os ambientes é baseada, principalmente, na análise de histórico de consumo e no custo associado na compra de energia elétrica de cada mercado. Nesse sentido, buscando auxiliar a tomada de decisão do consumidor potencialmente livre, ocorreu o desenvolvimento da ferramenta web APLAC, que analisa o histórico de consumo de energia do usuário e retorna um relatório contendo o melhor cenário possível de contração no ambiente livre considerando um portfólio de contratos bilaterais, que podem ser escolhidos pelo próprio usuário. Juntamente com esse cenário a ferramenta calcula o preço mínimo (preço de indiferença) podendo podendo viabilizar uma possível migração ao ambiente livre.

A ferramenta on-line está hospedada em uma máquina virtual, com o sistema operacional Ubuntu 16.04.2 LTS, que foi integrada ao servidor. As principais utilidades utilizadas pela máquina virtual para o funcionamento da ferramenta computacional são:
• AngularJS Framework;
• Bootstrap Framework;
• ChartJS Framework;
• Handsontable;
• mPDF;
• SGBD MariaDB;
• APIs implementadas por meio do Micro Framework Slim para que o usuário final possa obter as informações de forma rápida via AJAX (Asynchronous Javascript and XML);
• Algoritmos desenvolvidos em JavaScript;
• Algoritmos desenvolvidos em PHP;
• SGBD MariaDB para gerenciar os dados necessários, principalmente financeiros;

A ferramenta computacional está estruturada no formato dashboard e as informações provenientes dos dados estão apresentadas de forma gráfica, com a ajuda de plugins baseados em JavaScript, e que apresentem responsividade na aplicação. Dessa forma, a experiência proporcionada ao usuário será mais dinâmica e amigável. O desenvolvimento backend da ferramenta deve ser capaz de processar dados de consumo de energia elétrica, carga tributária, tarifas de uso, restrições contratuais, entre outros aspectos.

Em relação a otimização, foi selecioado o GLPK (GNU Linear Programming Kit), que é um MILPS (Mixed-Integer Linear Program Solver), usado para solução de problemas do tipo inteiro misto. O GLPK é compilado por meio do GNU Octave, que é software livre similar ao MATLAB. O GLPK e o Octave são iniciativas GNU, que busca desenvolver softwares de código aberto com a mesma qualidade de softwares de iniciativa privada.
